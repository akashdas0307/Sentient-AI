Sentient — GUI

This directory is the placeholder for the Phase 2 React frontend.

For the MVS, the backend serves an embedded minimal HTML console directly at http://127.0.0.1:8765/.

To build the React frontend in Phase 2:

  1. Scaffold with npm create vite@latest . -- --template react-ts
  2. Install Tailwind
  3. Implement Chat, Dashboard, HealthPanel, ModuleStatus components per ARCHITECTURE.md §9.3
  4. Connect to ws://localhost:8765/ws/chat and ws://localhost:8765/ws/dashboard
